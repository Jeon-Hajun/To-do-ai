# 로컬 개발 스크립트

## 주요 스크립트

### `start-dev.sh`
로컬 개발 환경을 시작합니다. 백엔드, AI 백엔드, 프론트엔드를 각각 별도의 터미널 창에서 실행합니다.

**사용법:**
```bash
./scripts/dev/start-dev.sh
```

**실행되는 서비스:**
- 프론트엔드: http://localhost:5173
- 백엔드: http://localhost:3000
- AI 백엔드: http://localhost:5000

### `start_all_services.sh`
모든 서비스를 시작합니다.

### `start_react.sh`
React 프론트엔드만 시작합니다.

### `install_and_start_frontend.sh`
프론트엔드를 설치하고 시작합니다.
