# 소스코드 분석 기반 PM 업무 자동화 플랫폼 - 차별성

## 핵심 가치 제안

**"소스코드로 프로젝트 진행도를 자동으로 파악하는 PM 업무 자동화 플랫폼"**

PM이 귀찮아하는 진행도 파악 업무를 GitHub 소스코드 분석으로 자동화합니다.

---

## 참신성

### 1. 소스코드 기반 진행도 자동 분석
- **기존**: PM이 수동으로 진행 상황 파악 및 보고
- **본 플랫폼**: GitHub 저장소 연동으로 커밋/이슈/브랜치 기반 자동 진행도 계산

### 2. Task-커밋 자동 연동
- Task와 GitHub 이슈 번호를 연결하면 자동으로 완료 상태 업데이트
- 커밋 메시지나 이슈 닫힘으로 Task 완료 자동 확인

### 3. AI 기반 Task 제안
- 코드 분석을 통해 누락된 작업이나 리팩토링 필요 영역 제안

## 타겟 시장

### Primary Target
- **중소기업 개발팀** (10-50명)
- **스타트업** (5-20명)
- 문제: 전문 PM 부재, 진행도 파악 어려움
- 해결: 자동화된 진행도 분석으로 PM 업무 경감

---

## 시연 포인트 (2주 프로토타입)

1. **GitHub 저장소 연결** → 자동으로 커밋/이슈 정보 수집 (30초)
2. **진행도 대시보드** → 코드 기반 진행도 자동 계산 및 시각화 (30초)
3. **Task-이슈 연동** → 이슈 닫힘으로 Task 자동 완료 (30초)
4. **AI Task 제안** → 코드 분석 기반 새 Task 제안 (30초)

---

## 기술적 우위

- **GitHub REST API** 활용으로 안정적인 데이터 수집
- **소스코드 기반 분석**으로 정확한 진행도 측정
- **AI 기반 자동화**로 PM 업무 효율화
