요구사항 상세 명세

S1-R01: 회원가입
- 행위자: 비회원
- 기능: 이메일, 비밀번호, 닉네임을 입력하여 회원가입
- 입력: 이메일, 비밀번호, 닉네임
- 출력: 회원가입 성공 시 사용자 정보 반환 (ID, 이메일, 닉네임)
- 제약사항: 이메일 중복 불가, 비밀번호 암호화 저장

S1-R02: 로그인
- 행위자: 비회원/회원
- 기능: 이메일과 비밀번호로 로그인하여 JWT 토큰 발급
- 입력: 이메일, 비밀번호
- 출력: JWT 토큰 및 사용자 정보 (ID, 이메일, 닉네임, 프로필 이미지)
- 제약사항: 비밀번호 검증 필수

S1-R03: 이메일 중복 확인
- 행위자: 비회원
- 기능: 회원가입 전 이메일 중복 여부 확인
- 입력: 이메일
- 출력: 사용 가능 여부 (true/false)

S1-R13: 로그아웃
- 행위자: 회원
- 기능: 현재 세션 종료 및 토큰 무효화
- 입력: 인증 토큰
- 출력: 로그아웃 성공 메시지

S1-R14: 회원 정보 수정
- 행위자: 회원
- 기능: 닉네임, 이메일, 비밀번호 변경 가능
- 입력: 변경할 정보 (닉네임, 이메일, 현재 비밀번호, 새 비밀번호)
- 출력: 수정된 회원 정보
- 제약사항: 현재 비밀번호 확인 필수

S1-R15: 프로필 이미지 업로드
- 행위자: 회원
- 기능: 프로필 이미지 파일을 업로드하여 저장
- 입력: 이미지 파일 (multipart/form-data)
- 출력: 업데이트된 프로필 이미지 파일명

S1-R16: 프로필 이미지 삭제
- 행위자: 회원
- 기능: 업로드한 프로필 이미지 삭제 및 기본 이미지로 복원
- 입력: 인증 토큰
- 출력: 기본 이미지로 변경된 프로필 정보

S1-R17: 회원 탈퇴
- 행위자: 회원
- 기능: 계정 및 관련 데이터 삭제
- 입력: 인증 토큰
- 출력: 탈퇴 완료 메시지
- 제약사항: 소유한 프로젝트가 있으면 삭제 불가 또는 관련 데이터 처리 필요

S2-R01: 프로젝트 생성
- 행위자: 회원
- 기능: 프로젝트 제목, 설명을 입력하여 새 프로젝트 생성
- 입력: 제목, 설명
- 출력: 생성된 프로젝트 정보 (ID, 제목, 설명, 소유자 정보)
- 제약사항: 생성자는 자동으로 프로젝트 소유자 및 멤버로 등록

S2-R02: 프로젝트 코드 생성
- 행위자: 회원
- 기능: 공유 프로젝트용 6자리 영숫자 코드 자동 생성
- 입력: 프로젝트 ID (생성 시)
- 출력: 생성된 프로젝트 코드
- 제약사항: 고유성 보장 필수

S2-R03: 프로젝트 코드 검증
- 행위자: 회원
- 기능: 프로젝트 코드의 유효성 확인
- 입력: 프로젝트 코드
- 출력: 코드 유효성 및 프로젝트 존재 여부

S2-R04: 프로젝트 참여
- 행위자: 회원
- 기능: 프로젝트 코드와 비밀번호로 공유 프로젝트 참여
- 입력: 프로젝트 코드, 비밀번호
- 출력: 참여 성공 및 프로젝트 정보

S2-R05: 프로젝트 목록 조회
- 행위자: 회원
- 기능: 내가 참여한 모든 프로젝트 목록 조회
- 입력: 인증 토큰
- 출력: 프로젝트 목록 (ID, 제목, 설명, 상태, 소유자 정보)

S2-R06: 프로젝트 상세 조회
- 행위자: 회원
- 기능: 프로젝트 ID로 상세 정보 조회
- 입력: 프로젝트 ID
- 출력: 프로젝트 상세 정보 (제목, 설명, 상태, GitHub 저장소, 멤버 정보 등)

S2-R07: 프로젝트 수정
- 행위자: 프로젝트 소유자
- 기능: 프로젝트 제목, 설명, 상태 수정
- 입력: 수정할 정보 (제목, 설명, 상태)
- 출력: 수정된 프로젝트 정보
- 제약사항: 소유자만 수정 가능

S2-R08: 프로젝트 삭제
- 행위자: 프로젝트 소유자
- 기능: 프로젝트 및 관련 데이터 삭제 (Task, 커밋 정보 등)
- 입력: 프로젝트 ID
- 출력: 삭제 완료 메시지
- 제약사항: 소유자만 삭제 가능, CASCADE 삭제

S2-R09: 프로젝트 멤버 조회
- 행위자: 회원
- 기능: 프로젝트 구성원 목록 조회
- 입력: 프로젝트 ID
- 출력: 멤버 목록 (ID, 닉네임, 역할, 참여일)

S2-R10: 프로젝트 멤버 삭제
- 행위자: 프로젝트 소유자
- 기능: 일반 멤버 제거
- 입력: 프로젝트 ID, 멤버 ID
- 출력: 삭제 완료 메시지
- 제약사항: 소유자는 삭제 불가

S2-R11: 프로젝트 탈퇴
- 행위자: 프로젝트 멤버
- 기능: 소유자가 아닌 일반 멤버만 프로젝트 탈퇴
- 입력: 프로젝트 ID
- 출력: 탈퇴 완료 메시지
- 제약사항: 소유자는 탈퇴 불가 (삭제만 가능)

S2-R12: GitHub 저장소 연결
- 행위자: 프로젝트 소유자
- 기능: GitHub 저장소 URL 및 Personal Access Token(Private 저장소용) 연결
- 입력: GitHub 저장소 URL, 토큰(선택)
- 출력: 연결 성공 및 저장소 정보
- 제약사항: 소유자만 연결 가능, 토큰은 암호화 저장

S3-R01: Task 생성
- 행위자: 프로젝트 멤버
- 기능: 프로젝트 내 작업 생성 (제목, 설명, 마감일, GitHub 이슈 번호 연결 가능)
- 입력: 프로젝트 ID, 제목, 설명, 마감일(선택), GitHub 이슈 번호(선택)
- 출력: 생성된 Task 정보 (ID, 제목, 설명, 상태, 생성일)
- 제약사항: 프로젝트 멤버만 생성 가능

S3-R02: Task 목록 조회
- 행위자: 프로젝트 멤버
- 기능: 프로젝트별 작업 목록 조회 (상태 필터 가능)
- 입력: 프로젝트 ID, 상태 필터(선택)
- 출력: Task 목록 (ID, 제목, 상태, 할당된 사용자, 마감일 등)

S3-R03: Task 상세 조회
- 행위자: 프로젝트 멤버
- 기능: Task ID로 상세 정보 조회
- 입력: Task ID
- 출력: Task 상세 정보 (제목, 설명, 상태, 할당된 사용자, 마감일, GitHub 이슈 번호 등)

S3-R04: Task 내용 수정
- 행위자: 프로젝트 소유자
