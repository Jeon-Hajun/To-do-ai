# 목표
안드로이드 앱 개발
- 웹뷰기반의 하이브리드 모바일 앱(모피어스)
- 웹 서비스 (모바일 화면 기준)
- 별도의 앱 구동
- 개발자 크롬 (chrome://inspect)
- AVD(10.0.2.2) / 실디바이스(127.0.0.1)\

**앱(웹뷰) 시작페이지 설정**
- Manifest.xml 파일에 아래의 부분 수정
```xml
<startpage>
    <!-- 아래의 부분을 적절하게 수정한다. 단말기를 기준으로 작성되여야한다. -->
    <name>http://10.0.2.2:5175</name>
</startpage>
```

**터미널에서 아래의 커맨드 입력**
1. adb devices (디바이스 연결 확인)
2. adb reverse tcp:5175 tcp:5175

**리액트 앱 (vite)**
- npm run dev (--host 옵션 추가)
- vite.config.json
   - server.proxy 리버스 프록시 설정
  
**Morpheus API**
- wiki.uracle.co.kr
- import/export 지원
- 앨범에서 사진가져와 업로드 예제를 스레드에서 제공.

**앱 이름**

`strings.xml` 파일에서 app_name 속성을가진태그 수정
```xml
<string name="app_name">SW인재양성</string>
```

**앱 아이콘 변경**
- [icon.kichen](http://icon.kitchen/) 아이콘 만들어서 다운로드 후 압축 풀기
- android/res 폴더내 @mipmap 폴더들을 androis/app/src/res 폴더로 붙여넣기
- AndroidManifest.xml 에서 아래와 같이 수정
```xml
<application
... 생략
android:icon="@mipmap/ic_launcher"
>
```
